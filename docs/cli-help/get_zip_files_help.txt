usage: get_zip_files.py [-h] [--datensaetze-only] [--election-zips-only] [--list-election-zips]
                        [--election-zip TOKEN] [--with-datensaetze]

Download Wahl-O-Mat ZIPs from bpb weitere Wahlen plus the Datensätze bundle, only the Datensätze
bundle, or a filtered subset of election ZIPs.

options:
  -h, --help            show this help message and exit
  --datensaetze-only    Download and extract only the Wahl-O-Mat-Datensätze bundle (no election
                        ZIPs).
  --election-zips-only  Download and extract only election ZIPs (no Datensätze bundle).
  --list-election-zips  Print election_id and URL for each ZIP on the weitere-Wahlen page, then
                        exit (no download). See docs/DATASET.md for what election_id means.
  --election-zip TOKEN  Substring to match against URL or election_id (repeat for several tokens).
                        Only matched archives are downloaded and extracted.
  --with-datensaetze    With --election-zip, also download and extract the Datensätze bundle.
